HTML-язык разметки гипертекста

Изучение языка HTML является отправной точкой для любого веб-разработчика.
Вся всемирная паутина построена на миллиардов HTML страниц ссылающихся друг на друга, поэтому знание этого языка святая обязанность каждого.

Смежные технологии

Но создание страниц для интернета web не заканчивается только на знание HTML.
Есть ещё ряд языков и технологий, без которых не обходится ни один полноценный сайт:

CSS
Каскадные таблицы стилей - используют для стилизации элементов HTML
JavaScript
Язык программирования обеспечивающий интерактивность веб-страниц

Этапы Изучения

При изучении всего Базового стека1 технологии веб-разработчика, рекомендуется соблюдать следующий порядок:

  1. HTML
  2. CSS
  3. JavaScript

В таком случае, информация будет усваиваться более просто, без информационных разрывов

Инструменты разработчика

Для создания простейшей HTML страницы нам могут потребоваться следующие инструменты:

  • Графическая операционная система
    • Windows
    • Ubuntu
  • Текстовый редактор
    • Блокнот
    • Vscode
  • Браузер для просмотра страниц
    • Chrome
    • Opera
